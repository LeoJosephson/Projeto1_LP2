# Projeto1_LP2
 Final Project for LP2
 
 **In this project, I will be developing an diagram software using only java**
 
 Features:
 
 -> Create shapes:
  * Rect (press r)
  * Ellipse (press e)
  * Text (press t)
  * Line (press l)
  * Triangle (press a)
  
-> Others:
* Delete object (press del)
* Grow object (press g)
* Move object (⬅, ➡, ⬆, ⬇)
* Select object (right/left click)
