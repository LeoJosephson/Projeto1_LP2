# Projeto1_LP2
 Final Project for LP2
 
 **In this project, I will be developing an diagram software using only java**
 
 Features:
 
 -> Shapes:
  * Rect (press r)
  * Ellipse (press e)
  * Text (press t)
  * Triangle (press a)
  * 
-> Others:
* Delete last shape (press del)
* Grow last shape (press g)
